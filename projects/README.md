# Projects

## Gradio

ปรับ `MODEL_PATH` และ `LOGO_PATH` ไปยัง path ที่ถูกต้อง จากนั้นรัน Gradio application ด้วยคำสั่ง:

``` sh
gradio webapp.py
```

หลังจากทีรันแล้วไปที่ `http://localhost:7860/` เพื่อใช้ web application
